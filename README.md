## Функционал, доступный в конфигурации

- Добавление сотрудника

- Добавление учебного года

- Добавление учебной организации

- установка времени обновления задач (для роли «Администратор»)

- Добавление пользователей в информационную базу

- Добавление отделения

- Добавление учебной группы

- Добавление студента

- Добавление предмета

- Добавление нагрузки пользователя

- Просмотр и создание индивидуальных настроек групп и предметов

- Формирование расписания

- Проведение занятия

- Формирование графиков учебного процесса

- Работа с журналом

- Формирование служебных записок

- Проведение аттестации

- Создание напоминалок (заметок)

- Добавление консультаций

- Работа с дипломниками

- Смена даты списка задач

- Получение статистики по успеваемости и посещаемости

- Просмотр выполненной нагрузки

- Получение общей статистики преподавателя

- Получение списка замен

- Получение списка совмещений (сдвоенных пар)

- Получение списка проведенных консультаций

- Анализ всех оценок по студенту

- Анализ пропусков студента, совместных с другими студентами

- Анализ проведенной аттестации на вероятность ее получения для группы

- Анализ объединения студентов в группы по успеваемости
